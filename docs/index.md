### Project Overview

Red Spotify is a cross-platform mobile application inspired by Spotify that enhances music discovery, listening analytics, profile customization, and library management using the Spotify Web API.

Built with **React Native + Expo**, the project integrates Spotify OAuth authentication and provides a modern user-focused streaming companion experience.

## Core Features

- Spotify OAuth Login  
- Dashboard with Listening Analytics  
- Top Songs / Top Artists  
- Playlist & Library Management  
- Search Tracks / Albums / Artists  
- Playback Controls  
- Profile Personalization  
- Cross Platform (iOS + Android)

---

## Technology Stack

- React Native  
- Expo  
- Node.js  
- Spotify Web API  
- OAuth 2.0  
- Render Backend Hosting  
- GitHub Version Control
